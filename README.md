## Présentation
Les expériences sont reproduisibles dans les notebook `exp1.ipynb` et `exp2.ipynb`. Les fichier `utils_exp1.py` et `utils_exp2.py` continnent le code (redondant) pour faire tourner les deux notebook et `flow_matching.py`contient le code de l'algorithm. Le tout est codé en Jax.

## Installation

Assurez-vous d'avoir Python==3.10 pour la compatiblité. Pour installer les dépendances nécessaires:
```bash
pip install -r requirements.txt
```
